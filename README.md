## 使用方法

1. 克隆此專案。
2. 在 `problems/` 資料夾中找到你感興趣的問題。
3. 運行測試：

```bash
go test ./problems/問題資料夾名稱/
```

例如：

```bash
go test ./problems/0001_two_sum/
```
